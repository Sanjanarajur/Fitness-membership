# 🏋️ DBMS Mini Project – Fitness Membership Management System

## 📌 Features

* Common home page for both **Customer** and **Admin**
* Customer **signup and login**
* Membership **registration form**
* Stores all member details in **MongoDB**
* **Admin dashboard** to view registered members
* Data retrieval using **MongoDB queries**

---

## 🛠️ Technologies Used

**Frontend:**

* HTML
* CSS
* JavaScript

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB

---

## ⚙️ Setup Instructions

1. Install **Node.js**
2. Open terminal in the project folder
3. Run:

   ```bash
   npm install
   ```
4. Create a `.env` file and add your MongoDB URI:

   ```
   MONGO_URI=your_mongodb_connection_string
   ```
5. Start the server:

   ```bash
   node server.js
   ```
6. Open `index.html` in your browser
